Welcome to Rethink.

Rethink is a css theme targeted at org mode. Rethink was inspired by its cousin Imagine and HTML5 UP's theme Story

Rethink is available on Github here. To use rethink add the following headers to your org document: 

#+HTML_HEAD: <link rel="stylesheet" type="text/css" href="rethink.css" />
#+OPTIONS: toc:nil num:nil html-style:nil
